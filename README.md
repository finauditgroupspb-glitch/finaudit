# FinAudit Group — корпоративный сайт

Многостраничный корпоративный сайт на **Next.js 16 + React 19 + TypeScript + TailwindCSS**. Интерфейс — на русском языке.

## Запуск

```bash
npm ci
npm run dev        # разработка → http://localhost:3000
npm run typecheck
npm run build
npm start          # production-сервер после сборки
```

Требуется Node.js 20.9 или новее.

## Что настроить перед публикацией

1. Замените `site.url` в `lib/site.ts` на реальный HTTPS-домен.
2. Заполните переменные окружения из `.env.example` и подключите доставку заявок через защищённый webhook/CRM или Telegram Bot API.
3. Разместите сайт за CDN/WAF или reverse proxy. Пример Nginx находится в `deploy/nginx.conf.example`.
4. Прочитайте `SECURITY.md` и не публикуйте origin-сервер напрямую в интернет.

## Основные команды

```bash
npm run typecheck
npm run build
npm run security:audit
```

## Структура

- `app/` — страницы и API-маршруты.
- `components/` — интерфейс, формы и эффекты.
- `lib/site.ts` — контакты и навигация.
- `lib/data.ts` — услуги, кейсы, статьи, вакансии и FAQ.
- `deploy/nginx.conf.example` — пример защитного reverse proxy.
- `SECURITY.md` — требования к защите и инфраструктуре.

## Важно перед публикацией

Кейсы, отзывы, статистика, юридические страницы и тексты блога необходимо сверить с реальными данными компании и действующим законодательством. Не публикуйте демонстрационные данные как подтверждённые факты.

Шрифт Manrope хранится локально в проекте через `@fontsource-variable/manrope`; production-сборка больше не зависит от Google Fonts.

## Как изменить номер телефона

Номер телефона централизован в файле `lib/site.ts`.

Найдите строку:

```ts
const contactPhone = "+7 (983) 096-57-86";
```

Замените только значение номера, например:

```ts
const contactPhone = "+7 (999) 123-45-67";
```

После сохранения новый номер автоматически изменится в шапке, подвале, на странице контактов, в мобильном меню, плавающей кнопке и микроразметке сайта. Ссылка для звонка сформируется автоматически.
